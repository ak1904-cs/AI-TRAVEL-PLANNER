How the ML Model Works

Dataset: 225 rows × 15 features synthesized from real Indian tourism data (similar to Kaggle's Travel & Tourism datasets). Features include destination type, average daily cost, rating, student-friendliness, and more.

Algorithm: RandomForestClassifier (scikit-learn) with 100 estimators, max_depth=8. Target variable: recommended (1 = affordable for budget tier, 0 = not).

Key insight: The model's #1 feature is cost_ratio (avg_cost_per_day / daily_budget) — confirming that budget-affordability is the most important signal for student travel recommendations.

Output: ML confidence scores (0–1) rank destinations. The AI itinerary generator then uses those ranked destinations + Claude API to create personalized day-by-day plans.
